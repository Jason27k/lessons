# Java Basics - Lesson Plan

## Goal
Build a solid foundation in Java by understanding the structure of a Java program and the core building blocks used in every program.

## Lessons

### 1. Your First Java Program
- The `class` keyword
- The `main` method signature
- `System.out.println()` and `System.out.print()`
- Packages

### 2. Variables and Data Types
- Declaring and assigning variables
- Primitive types: `int`, `double`, `boolean`, `char`
- The `String` type
- `var` for type inference (Java 10+)

### 3. Arithmetic Operators
- `+`, `-`, `*`, `/`, `%`
- Integer division vs floating-point division
- Increment and decrement: `++`, `--`
- Compound assignment: `+=`, `-=`, `*=`, `/=`

### 4. If Statements
- `if`, `else if`, `else`
- Comparison operators: `==`, `!=`, `<`, `>`, `<=`, `>=`
- Logical operators: `&&`, `||`, `!`
- Nested if statements

### 5. For Loops
- Basic `for` loop
- `while` loop
- `do-while` loop
- `break` and `continue`

### 6. Methods
- Defining methods: return type, name, parameters
- `void` vs returning a value
- Parameters vs arguments
- Early `return`
- Method overloading
- Pass by value
